#  Floricultura - Projeto Full Stack

##  Sobre o Projeto
Este projeto consiste em uma aplicação full stack de uma floricultura, onde é possível visualizar, cadastrar, editar e excluir produtos (flores).
O sistema foi desenvolvido com foco em um catálogo de produtos, simulando um e-commerce simples.

---

##  Tecnologias Utilizadas

###  Backend
- Node.js  
- Express  
- MySQL  

###  Frontend
- HTML  
- CSS  
- JavaScript  

###  Banco de Dados
- MySQL
  
---

##  Funcionalidades

-  Listar produtos  
-  Cadastrar flores  
-  Editar produtos  
-  Excluir produtos
